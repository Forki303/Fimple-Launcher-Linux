# Fimple Launcher Linux

## Русский

Fimple Launcher — простой и удобный лаунчер для Minecraft на Linux.  
На данный момент версия для Linux находится в бета, поэтому может работать менее стабильно, чем версия для Windows.

С ним вы сможете:  
- Выбирать и запускать любую версию Minecraft.  
- Настраивать лаунчер через простой интерфейс.  
- Играть без лишних сложностей и долгих настроек.

### Установка

1. Скачайте архив .tar.gz с лаунчером.  
2. Распакуйте архив:  
   
   tar -xzvf FimpleLauncherLinux.tar.gz(version)

3. Перейдите в папку лаунчера:

cd FimpleLauncherLinux(version)


4. Сделайте исполняемый файл скрипта:

chmod +x FimpleLauncher


5. Запустите лаунчер:

./FimpleLauncher

## English

Fimple Launcher — a simple and convenient launcher for Minecraft on Linux.
Currently, the Linux version is in beta, so it may be less stable than the Windows version.

With it, you can:

Choose and launch any Minecraft version.

Customize the launcher through a simple interface.

Play Minecraft quickly without extra hassle.


Installation

1. Download the .tar.gz archive with the launcher.


2. Extract the archive:

tar -xzvf FimpleLauncherLinux.tar.gz(version)


3. Go to the launcher folder:

cd FimpleLauncherLinux(version)


4. Make the main script executable:

chmod +x FimpleLauncher


5. Run the launcher:

./FimpleLauncher
